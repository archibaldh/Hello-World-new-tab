## Hello World New Tab 

Hello world new tab is a Minimalist and Open Source New Tab chromium extension. This extension works on any Chromium based browser such as Brave, Edge, Chrome or Opera.

You can view the code [here](https://github.com/archibowl/Hello-World-new-tab). Instructions to install are down below or in the readme.md file in the repository.
### Installation

Hello World New Tab is not availabe on the chrome webstore yet and you must install it as an unpacked extension. To do this, follow the instructions below.

1. Install the source code as a .zip file.
2. Extract the .zip file.
3. Go to chrome://extensions/
4. Turn on Developer mode. 
5. Click load unpacked.
6. A file manager window should appear. Navigate to your folder with the source code, highlight it and click "Select Folder"

Congratulations, you have just installed hello world new tab!
